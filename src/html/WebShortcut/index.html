This website is still in the Alpha phase. There are a number of issues which are listed below. However, feel free to use it and please let me know of any feedback you may have.

This website allows you to launch and to convert web shortcut files. Web shortcuts are the files that are created by various operating systems when you save a link to a website. It supports shortcut files for Windows, Linux, and Apple. Specifically, these are files with the following extensions: .url, .website, .desktop, .webloc . This website is part of the Web Shortcut Utility Suite.

The source code for the website is available here.

Known issues

  • When converting shortcuts with unicode characters, the downloaded ZIP file will be corrupt. As a workaround, download the files individually.
  • On the Android browser, the website will report an error for files that have unicode characters in the file name. This requires more investigation, but appears to be an issue with the browser itself.

Privacy

Not only are the shortcut files uploaded to a server, but they are sent unencrypted. If you have any privacy concerns, I suggest that you avoid this website.
